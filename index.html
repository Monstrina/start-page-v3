Настройка значков

Эта стартовая страница использует Font Awesome. Вы можете настроить значки для категорий и ссылок в Настройках.

Шаг 1: Найдите значок

  1. Перейдите на fontawesome.com/search
  2. Введите ключевое слово в поле поиска (например, "linux", "music", "cloud")
  3. Нажмите кнопку фильтра "Free" справа, чтобы отобразить только бесплатные значки
  4. Нажмите на значок, который хотите использовать

Шаг 2: Скопируйте код значка

  1. После нажатия на значок появится всплывающее окно
  2. Найдите HTML-код, который выглядит так: <i class="fa-brands fa-linux"></i>
  3. Скопируйте только значение класса — часть в кавычках
  4. Например, скопируйте: fa-brands fa-linux

Шаг 3: Использование в настройках

  1. Откройте Настройки и перейдите на вкладку Категории или Ссылки
  2. Вставьте код значка в поле значка (первое поле ввода)
  3. Вы увидите, как появится предварительный просмотр вашего значка

Примеры

  • fa-solid fa-code → (сплошной стиль)
  • fa-brands fa-github → (логотип бренда)
  • fa-solid fa-gamepad →
  • fa-brands fa-spotify →

Совет: Используйте fa-solid для общих значков или fa-brands для логотипов компаний/брендов.
